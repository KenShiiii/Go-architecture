## git command
initialize repository
```
git clone
```
version tags
```
git tag v0.0.1
git push --tags
```
view change log
```
git log
```

You can pull down different versions of the code with

```
git checkout v0.0.1
```

###**remember to checkout master when you’re done**
```
git checkout main
```
